Computer Lab-3


Computer Lab 3 is a dedicated facility for students specializing in Data Science, providing them with high-performance computing resources, advanced data analytics tools, and cloud-based platforms. The lab is designed to support data processing, machine learning, big data analytics, and visualization, helping students gain hands-on experience in real-world data-driven applications.

Lab Features & Infrastructure

✅ High-Performance Computing Systems optimized for data processing and machine learning.
✅ Big Data & Cloud Integration with access to Google Cloud, AWS, Azure, and Databricks.
✅ Pre-installed Data Science & Analytics Software, including:

Programming Languages: Python, R, SQL
Data Processing & Analysis: Pandas, NumPy, SciPy
Big Data Technologies: Hadoop, Spark, Kafka
Machine Learning Frameworks: TensorFlow, Scikit-learn, PyTorch
Data Visualization Tools: Tableau, Power BI, Matplotlib, Seaborn
Database Management: MySQL, PostgreSQL, MongoDB

Purpose & Usage

Data Analytics Training: Supports data cleaning, preprocessing, statistical modeling, and predictive analytics.
Big Data & Cloud Computing: Enables students to work on large-scale datasets using Hadoop, Spark, and cloud platforms.
Machine Learning & AI Applications: Facilitates model development, deep learning, and NLP projects.
Research & Industry Collaboration: Provides a platform for real-world projects, hackathons, and workshops with industry experts.

Guidelines & Access

Open for Data Science students during academic hours with extended access for research projects.
Supervised by faculty members and lab assistants to ensure best practices.
Adherence to ethical data usage policies and responsible computing.
